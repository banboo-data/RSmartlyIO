Discovered any bugs?

  1. Please make sure you run with the latest RSmartlyIO version before you report any issues.

  2. Do you have a valid API token?

  3. If you are still in trouble, please report the issue on https://github.com/rstats-lab/RSmartlyIO/issues

We appreciate if you provide the following information along with the issue:

  • Operating System
  • R Version
  • RSmartlyIO version
  • the R code which leads to the error
  • the error message, prompt output
  • code as text rather than screenshots

We will try to get back to you as soon as possible!
